# kejian_enterprise_Api

基于node.js与Koa构建的本地接口，用于 kejian_enterprise 项目：[HZhertz/Kejian_enterprise 基于Vue2和elementUI的企业官网](https://github.com/HZhertz/Kejian_enterprise)

## Project setup

### install

```
// npm
npm install

// pnpm
pnpm install
```

### Run serve

```
// npm
npm run serve

// pnpm
pnpm run serve
```

## 技术栈

1. koa                           构建web服务
2. koa-router                路由接口
3. mysql2                     连接数据库
4. koa-static                 静态资源
5. koa2-cors                 解决跨域
6. @koa/multer            文件上传
7. jsonwebtoken           jwt鉴权
8. bcryptjs                    密码加密
